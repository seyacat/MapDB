# MapDB

MapDB is a tool for managing relationships between objects like a database does.

## Instalación

```
npm i --save @seyacat/mapdb
```
## Usage

Import in project
```
const { MapDB } = require("@seyacat/mapdb");
const mdb = new MapDB();
```
as module
```
import { MapDB } from "@seyacat/mapdb";
const mdb = new MapDB();
```

## Test

Integrated tesr with mocha
```
npm run test
```
